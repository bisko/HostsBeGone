# HostsBeGone!

### What's that?

HostsBeGone is a local DNS server, written in NodeJS with a web-based GUI based on React.

### Prerequisites 

* macOS 10.11+
* NodeJS 6.9+

### Installing

```
$ git clone https://github.com/bisko/HostsBeGone.git hostsbegone
cd hostsbegone
npm install
```

### Running

```
sudo node start.js
```

### To stop it from working

```
Ctrl + C
```
